Плагин выводит связанные записи к постам с миниатюрами этих записей.

== Description ==

Плагин выводит связанные записи к постам на основе меток (тегов), либо на основе категорий - метод привязки определяется в настройках.

Помимо этого, формат вывода может быть как вертикальным, так и горизонтальным. Вывод миниатюр и описания можно отключить.
  
Более подробную информацию вы найдете на [странице плагина](https://obg.kz/novyj-plagin-vyvoda-svyazannyx-zapisej-ark-relatedpost.html).

Если вам понравился мой плагин, то `пожалуйста` поставьте ему 5 звезд.


== Installation ==

1. Загрузите плагин ark_relatedpost на свой блог
2. Активируйте его
3. Настройте по своему усмотрению в панели управления на странице `Настройки - Связанные записи`.
4. Более подробную информацию о плагине Вы можете найти на [странице плагина](https://obg.kz/novyj-plagin-vyvoda-svyazannyx-zapisej-ark-relatedpost.html).

== Frequently Asked Questions ==

= Может ли плагин выводить записи по категориям? =

Да. Вывод связвнных записей может осуществляться как по категориям, так и по меткам.

= Можно ли изменить размер миниатюры? =

Можно. Это легко делается в настройках плагина.

= Оптимизирован ли вывод связанных записей под мобильные устройства? =

Оптимизирован. Такая оптимизация внедрена в версии 2.7.
